# 📚 Libralia (Spring Boot Edition)

Libralia es una aplicación de biblioteca personal con enfoque social, desarrollada como proyecto formativo mientras avanzo en mi formación como desarrolladora backend en el programa ONE - Oracle Next Education | Alura Latam.

Este proyecto es la evolución directa de mi primer repositorio [Libralia en Java Puro](https://github.com/NattyPavez/mi-proyecto-libralia), donde construí toda la lógica base desde cero.  
A medida que avanzo en el curso de *Java: trabajando con lambdas, streams y Spring Framework*, he migrado manualmente el código, incorporando nuevas herramientas del ecosistema Spring y aplicando mejores prácticas en cada etapa.

---

## 🚀 Funcionalidades actuales

- Crear perfil personalizado de usuario con nombre, edad, descripción y redes sociales.
- Buscar libros por título o autor desde la API pública de Google Books.
- Agregar libros a tu lista de *leídos*, *por leer* o simplemente ignorarlos.
- Visualizar tu perfil con lectura actual y libros almacenados.
- Interacción fluida por consola, validaciones dinámicas, y flujo organizado.
- Búsquedas inteligentes priorizando coincidencias exactas.
- Lógica modular que evita duplicaciones y facilita la mantenibilidad.

## 🧠 Conceptos aplicados en este curso

Durante esta etapa del desarrollo implementé los siguientes conocimientos:

## 🎯 Visión: experiencia social de lectores.

Libralia no es solo una app de biblioteca. Es un espacio pensado para que los lectores puedan conectar a través de sus intereses, compartir sus opiniones, reseñas, fotos y emociones lectoras.

Mi objetivo es construir una experiencia lectora que combine:

- 📘 Rigurosidad técnica en el desarrollo backend
- 🧩 Aplicación de lógica instintiva
- 💬 Interacción social con otros lectores  
- ✍️ Reseñas, debates y registros personalizados de lectura  
- 📸 Elementos multimedia para compartir visualmente la experiencia  
- 📊 Herramientas que ayuden a conocerse mejor a través de los libros leídos  

Este proyecto está en constante evolución, al igual que yo como desarrolladora. Cada nueva funcionalidad será un paso para mi formación.
